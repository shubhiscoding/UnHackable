## Getting Started

First, create a .env file
```bash
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
NEXTAUTH_URL=
AUTH_SECRET=
DATABASE_URI=
NEXT_PUBLIC_RPC_URL=
```
perform ```npm install --legacy-peer-deps```

then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
